# 🧠 Mental Health Impact Analysis using Machine Learning Regression

## 📌 Project Overview
Mental Health Impact Analysis is a Machine Learning project that predicts mental health impact levels based on various factors such as work pressure, academic stress, lifestyle habits, sleep patterns, and personal well-being indicators. The project uses regression techniques to analyze relationships between these factors and mental health outcomes.

## 🎯 Objectives
- Analyze factors affecting mental health.
- Predict mental health impact scores using regression models.
- Identify key variables contributing to mental health challenges.
- Support data-driven decision-making for mental health awareness.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

## 🔍 Methodology
1. Data Collection & Exploration
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Model Training using Regression
6. Model Evaluation
7. Prediction and Visualization

Organizations and educational institutions can use this system to understand factors affecting employee or student well-being. The insights can help improve workplace policies, reduce stress-related issues, enhance productivity, and support mental health initiatives.
